# Line chart
資料來源：[政府開放資料](http://data.gov.tw/)
資料名稱：空氣品質監測日污染指標
資料型式：曲線圖
動畫：文字位移、曲線位移、座標軸度量變化
說明：資料進場時會有動畫，如果取消勾選某地區，則資料會以動畫同式出場。點選下方月份便可以觀看該月資料，繼續點選下個月份，就可以不斷往下看，直到沒資料，則標籤會變紅色。
資訊：明顯看出東部空氣品質較西部好。
